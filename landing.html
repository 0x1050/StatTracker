This is just a template page. It includes all of the elements that we would pobably use to in the presentation of our project. I think the best idea would be to create separate css files with different theme names, and have javascript get rid of the current one and import the new one with asynchronously when we chose the name of it from the submenu up top.

Although the actual pages that we create for the project will be a lot less simpler than what we have here, this gives us a good foundation to build upon. I would recommend checking out fonts.google.com for different fonts to use. The easiest way to use them is to click on the top where it says to use the old style. This allows you to add fonts to a stylesheet. After you choose your fonts, you can click on where it says "#fonts selected" and grab the address in the href. Pasting this address into your url will allow you to copy the entire css file. From here, the best thing to do is to create a file within the css folder of the project with the name in the form of [themename]fonts.css. This will allow you to use @import url("[themename]fonts.css within your css file. Or you can just paste it in. Whatever you want.

Besides that, you can check w3schools.com if you need help with learning basic CSS. Their tutorials are pretty simple, but contain the essential concepts you would need to brush up on CSS. I personally love this website, since it is where I learned HTML and CSS many years ago. You can also learn JavaScript, MySQL, and PHP there as well. I can't stress enough how awesome this site is.
